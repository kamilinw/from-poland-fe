## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run dev

# watch mode
$ npm run build

# production mode
$ npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Run tests

```bash
# unit tests
$ npm run test

# unit tests with coverage report
$ npm run test:cov
```
